# Pull with rebase

| Pull with rebase for . . .           | Command                                |
| ------------------------------------ | -------------------------------------- |
| all the current user's repositories. | `git config --global pull.rebase true` |
| only the current repository.         | `git config pull.rebase true`          |
| only the current pull.               | `git pull --rebase`                    |
